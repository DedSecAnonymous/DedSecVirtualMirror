Packageflare.physics.colliders
Classpublic final class BoxCollider
InheritanceBoxCollider Collider Object

A Box collider object. A collider defines the main behavior and shape of each object in the scene. By default, a collider behaves as a kinematic object, which means they can be manually moved or be moved by other objects but no forces are generated from these actions, objects simply push each other until there is no overlaping between them. A collider is Kinemaic unless it is set to be static, rigid body or trigger.
